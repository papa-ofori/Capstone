Block Sites Overview

Block Sites on a router is a feature that allows the router to block access to specific websites or web services by either an IP address or a domain name. “Block Sites” is sometimes referred to as Parental control. This feature is often used to restrict access to certain types of content, such as adult websites or social media platforms, for various reasons. Site blocking is achieved using a filtering system that checks incoming and outgoing traffic against a blacklist of banned websites. If a user tries to access a blocked site, the router will deny the request and prevent the user from accessing the content.

How to Block Sites

  • Access the web interface of the router.
  • Locate and select “block sites”, “Parental Control” o “Access Restriction” of the left pane of the web interface.
  • Enter the domain name, URL, or a keyword of the website you want to block in the text box provided.
  • Click “Apply” or “Save.”
  • Reboot the router.

Tips

Here are a few additional tips to keep in mind when blocking sites:

  • Set up time restrictions: You can also set up time restrictions to limit your child's internet access during certain hours, such as bedtime or homework time. You also block sites to preserve bandwidth when children are away for school.
  • Enable logs: A log is a detailed record of the Web sites that users on your network have accessed or attempted to access. If you have set up content filtering on the Block Sites page, the Logs page shows you when someone on your network tried to access a blocked site.
